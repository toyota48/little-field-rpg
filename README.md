# 豊田の逆襲

HTML、CSS、JavaScriptだけで動作する、ブラウザ用の小さな2D RPGです。外部ライブラリは使用していません。

## 起動方法

このリポジトリのフォルダ（例: `/Users/tetsushi44/Documents/RPG_test`）でローカルサーバーを起動します。

```sh
python3 -m http.server 8000
```

ブラウザで以下を開きます。

```text
http://localhost:8000/
```

`index.html` を直接ブラウザで開いても動作します。

### スマホからアクセスする場合

Macとスマホを同じWi-Fiに接続した状態で、Mac側でサーバーを起動します。

```sh
python3 -m http.server 8000 --bind 0.0.0.0
```

MacのWi-Fi用IPアドレスを確認します。

```sh
ipconfig getifaddr en0
```

スマホのブラウザでは、`localhost` ではなくMacのIPアドレスを使って開きます。

```text
http://MacのIPアドレス:8000/
```

例:

```text
http://192.168.1.23:8000/
```

macOSのファイアウォール確認が出た場合は、Pythonの着信接続を許可してください。

## 操作方法

- 移動: PCでは矢印キーまたは WASD キー、スマホでは画面の十字キー
- スマホの十字キーは長押しで連続移動できます。
- 決定、会話、宿屋利用: Enter、Space、Z、E、または画面の「決定」ボタン
- 戦闘トップ: 「たたかう」「どうぐ」「ぼうぎょ」「逃げる」
- たたかう: 「攻撃」「特技」「魔法」を選択
- 特技: レベルに応じて「全力斬り」「二段斬り」「かぶと割り」を習得します。
- 魔法: レベルに応じて「回復」「メラ」「ベホイミ」を習得します。
- 特技、魔法、どうぐは、対象のコマンドを選んだあとに使用確認が表示されます。
- 戦闘ショートカット: トップでは 1 がたたかう、2 がどうぐ、3 がぼうぎょ、4 が逃げる。たたかう内では 1 が攻撃、2 が特技、3 が魔法、4 または Esc が戻る。特技/魔法/どうぐリストでは 1、2、3 で候補を選びます。
- セーブ、ロード: PCでは画面右上、スマホでは「メニュー」内の「セーブ」「ロード」ボタン
- 「目的OFF/ON」でフィールド上の現在目的表示を切り替えられます。
- スマホのメニューでは「ステータス」「アイテム」「冒険の記録」タブを切り替えられます。
- アイテムは「装備品」「道具」「大切なもの」に分かれ、装備、使用、破棄を行えます。大切なものは捨てられません。
- 「旅の地図」は「アイテム」→「大切なもの」から選び、「見る」で確認できます。

## ゲーム内容

- 第1章「閉ざされた東の道」を最後まで進められます。
- 始まりの町、北の草原、迷いの森、森の奥地、岩山への道、岩山の洞窟1階、岩山の洞窟地下1階、洞窟最深部、港町のマップがあります。
- 町ではNPCと会話でき、宿屋でHPとMPを全回復できます。道具屋、装備屋、入れる民家もあります。
- チュートリアル後、町の村人から「旅の地図」を受け取れます。
- 旅の地図では、まだ訪れていない地域は「？？？」として表示されます。
- メニューの「冒険の記録」では、現在の目的と達成済みの目的を確認できます。
- NPC会話、宝箱、宿屋などのメッセージは、フィールド画面上のメッセージウィンドウに表示されます。
- 初めて開始したときは、ルカによる会話と戦闘チュートリアルが始まります。
- フィールドは画面より大きいマップになっており、プレイヤーの移動に合わせてフィールド表示がスクロールします。
- 草原、森、岩山、洞窟では歩いているとランダムに敵と遭遇します。マップごとに出現する敵と出現率が変わります。
- 草原や洞窟には調べられる石碑、隠し通路、壊せる岩、隠し宝箱があります。
- 歩数によって昼夜が切り替わり、一部NPCの立ち位置が変わります。
- 森の奥地と洞窟最深部には、話しかけると会話後に戦闘になるボスがいます。
- 森のボスを倒すまでは東の道が閉ざされ、倒すと岩山方面へ進めるようになります。
- 岩山の洞窟には、鍵、レバー、回復の泉、複数階層の攻略要素があります。
- 港町に初めて到着すると、第1章クリアのイベントが発生します。
- 戦闘はレトロRPG風の黒い戦闘画面で進みます。
- 戦闘は「たたかう」「どうぐ」「ぼうぎょ」「逃げる」を選ぶターン制です。
- 「たたかう」から通常攻撃、特技、魔法を選びます。
- ボスは強攻撃の前に予兆を出します。「ぼうぎょ」でそのターンの被ダメージを軽減できます。
- ボスはHPが半分以下になると第二形態に移行し、行動パターンが変化します。
- 命中率、回避率、会心率が戦闘計算に反映されます。
- 攻撃計算には攻撃力、防御力、特技補正、装備補正、器用さ補正、会心率を使います。
- 素早さと器用さの補正で行動順が決まります。ただし1ターン中に同じ側が2回行動することはありません。
- 会心率は器用さで補正されますが、最大でも約7%に収まるようにしています。
- 「どうぐ」では薬草、上薬草、魔法の水、毒消し草、目覚めの花、しびれ消し草を使えます。使える道具が戦闘中の候補に優先表示されます。
- 毒、眠り、麻痺、火傷の状態異常があります。毒と火傷はターン終了時にダメージを受け、宿屋や回復ポイントで解除できます。
- 炎属性の魔法や属性武器には、敵ごとの弱点や耐性が反映されます。
- 一部の装備には属性、回避率、会心率、毒耐性などの特殊効果があります。特定の装備を揃えるとセットボーナスが発動します。
- 各地の宝箱から道具、お金、装備、大切なものを入手できます。装備は入手時に自動装備されます。
- 戦闘開始、勝利、攻撃、会心、弱点、状態異常、宝箱、岩破壊、セーブ、レベルアップ時は、ブラウザ標準の Web Audio API で生成したSEが再生されます。
- BGMも Web Audio API で生成しています。タイトル、町、フィールド、森、洞窟、通常戦闘、ボス戦、港町で曲調が切り替わります。
- ブラウザの自動再生制限に対応するため、BGM/SEは最初のクリックやキー入力後に鳴り始めます。「音をオン」ボタンを押すと確認音が鳴り、ボタン表示が「音: ON」に変わります。
- 敵を倒すと経験値とお金を獲得します。
- 経験値が一定値に達するとレベルアップし、最大HP、最大MP、攻撃力、防御力、賢さ、素早さ、器用さが上がります。
- レベルアップ時のHP/MP回復は、最大値が上がった分だけ回復します。
- 対象レベルに到達すると新しい特技や魔法を覚えます。
- セーブデータは `localStorage` に保存されます。ストーリーフラグ、訪問済みマップ、宝箱、ボス撃破、洞窟ギミック、状態異常、目的表示設定も保存されます。
- 古いセーブデータは、足りない項目を補完して読み込みます。

## ファイル構成

```text
RPG_test/
├── index.html   # 画面構造
├── styles.css   # レイアウトと見た目
├── game.js      # ゲーム状態、描画、入力、戦闘、マップ、クエスト、セーブ処理

```

## 拡張しやすい箇所

- マップ追加: `game.js` の `MAPS` にタイル配列、出口、NPC、中ボスを追加します。
- 敵追加: `game.js` の `ENEMY_TEMPLATES` に敵データを追加し、各マップの `encounters` に敵IDを追加します。
- ストーリー追加: `DEFAULT_GAME_FLAGS`、`QUEST_STEPS`、NPCの `dialogues`、出口の `requiredFlags` / `blockedMessage` を追加します。
- 宝箱追加: `MAPS` 内の `entities` に `kind: "chest"` と固有 `id`、`contents` を追加します。
- タイル追加: `game.js` の `TILES` と `drawTile()` に定義を追加します。
- アイテム追加: `TOOL_ITEMS`、`KEY_ITEMS`、`inventory` の構造を参考に、分類、説明、効果を追加します。
- 装備追加: `game.js` の `EQUIPMENT_ITEMS` に装備データを追加し、宝箱の `contents.equipment` に装備IDを指定します。
- 特技・魔法追加: `game.js` の `BATTLE_ACTION_DETAILS` に `category`、`minLevel`、`mpCost`、効果パラメータを追加します。
