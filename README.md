# Little Field RPG

HTML、CSS、JavaScriptだけで動作する、ブラウザ用の小さな2D RPGです。外部ライブラリは使用していません。

## 起動方法

このフォルダでローカルサーバーを起動します。

```sh
python3 -m http.server 8000
```

ブラウザで以下を開きます。

```text
http://localhost:8000/
```

`index.html` を直接ブラウザで開いても動作します。

### スマホからアクセスする場合

Macとスマホを同じWi-Fiに接続した状態で、Mac側でサーバーを起動します。

```sh
python3 -m http.server 8000 --bind 0.0.0.0
```

MacのWi-Fi用IPアドレスを確認します。

```sh
ipconfig getifaddr en0
```

スマホのブラウザでは、`localhost` ではなくMacのIPアドレスを使って開きます。

```text
http://MacのIPアドレス:8000/
```

例:

```text
http://192.168.1.23:8000/
```

macOSのファイアウォール確認が出た場合は、Pythonの着信接続を許可してください。

## 操作方法

- 移動: PCでは矢印キーまたは WASD キー、スマホでは画面の十字キー
- 決定、会話、宿屋利用: Enter、Space、Z、E、または画面の「決定」ボタン
- 戦闘: 画面の「攻撃」「回復」「逃げる」ボタン
- 戦闘ショートカット: 1 が攻撃、2 が回復、3 が逃げる
- セーブ、ロード: 画面右上の「セーブ」「ロード」ボタン

## ゲーム内容

- 町と草原の2つのマップがあります。
- 町ではNPCと会話でき、宿屋でHPを全回復できます。
- 草原では歩いているとランダムに敵と遭遇します。
- 戦闘は「攻撃」「回復」「逃げる」を選ぶターン制です。
- 戦闘中やレベルアップ時は、ブラウザ標準の Web Audio API で生成したSEが再生されます。
- 敵を倒すと経験値とお金を獲得します。
- 経験値が一定値に達するとレベルアップし、最大HPと攻撃力が上がります。
- セーブデータは `localStorage` に保存されます。
